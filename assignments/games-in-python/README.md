
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

In this exercise you'll build the classic Hangman word-guessing game using Python. The goal is to practice working with strings, loops, conditionals, and user input while creating a playable program.

## 📝 Tasks

### 🛠️ Hangman Core Logic

#### Description

Implement the core mechanics of the Hangman game. This includes selecting a word at random and processing user guesses letter-by-letter.

#### Requirements

Completed program should:

- Randomly select a word from a predefined list stored in your code
- Prompt the user to guess a single letter at a time
- Reveal correct letters in the hidden word in an `_ _ _` style format
- Track and display the number of incorrect guesses remaining
- Terminate with a win message when the user guesses the entire word
- Terminate with a lose message when the user runs out of attempts

### 🛠️ Optional Enhancements

#### Description

Once the core game works, consider adding extra features to improve user experience.

#### Requirements

- Allow the player to guess the full word
- Prevent duplicate guesses from penalizing the player
- Display a simple ASCII-art hangman that progresses with each wrong guess
- Let the player restart the game without rerunning the script
